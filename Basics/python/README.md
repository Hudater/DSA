# DSA Basics — Phase 0

Core data structures implemented from scratch in Python.
The goal is to understand internals before jumping into Neetcode 150 problems.

## Files

| File             | Covers                                                 |
| ---------------- | ------------------------------------------------------ |
| `linked_list.py` | Node, insert head/tail, delete, traverse, reverse      |
| `stack_queue.py` | Stack (LIFO), Queue via two stacks (FIFO)              |
| `binary_tree.py` | BST insert, BFS (level order), DFS (in/pre/post order) |
| `hashmap.py`     | Hash function, chaining for collision handling         |

## How to run

```bash
python linked_list.py
python stack_queue.py
python binary_tree.py
python hashmap.py
```
