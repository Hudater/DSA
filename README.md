# DSA

Solving [Neetcode 150](https://neetcode.io/practice) problems. Tracking progress from absolute basics to interview-ready.

---

## Structure

```txt
DSA/
├── Basics/
│   ├── Uni-c/              # university implementations (C)
│   └── python/             # scratch implementations
│       ├── linked_list.py
│       ├── stack_queue.py
│       ├── binary_tree.py
│       └── hashmap.py
├── LeetCode/
│   ├── arrays/
│   ├── two_pointers/
│   ├── sliding_window/
│   ├── stacks/
│   ├── binary_search/
│   ├── linked_lists/
│   ├── trees/
│   ├── graphs/
│   ├── heaps/
│   └── greedy/
└── XOR/
```

Each LeetCode problem folder contains:

- `solution.py` — accepted solution with approach comment
- `question.md` — problem statement + link
